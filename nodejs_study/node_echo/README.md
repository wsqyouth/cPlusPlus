### 一、创建工程布局
```
./
├── bin
│   └── node_echo // 可执行文件
├── doc // 文档
├── lib // API lib库
│   └── echo.js
├── node_modules // 第三方包
│   └── argv
├── package.json // 元数据文件
├── package-lock.json
├── README.md
└── tests
```
### 二、操作命令
```
npm install argv
node bin/node_echo hello coopers
```
